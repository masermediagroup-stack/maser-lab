# Lab app

Next.js shell for building and previewing **all Maser Media web UI** — page sections, components, forms, navigation, scroll reveals, micro-interactions, and Three.js systems.

**Agent instructions:** see the repository root [`../AGENTS.md`](../AGENTS.md). Load `maser-lab-web` for any web UI work; load `maser-lab-threejs` for canvas/WebGL/Three.js.

```bash
npm run dev    # http://localhost:3000
npm run lint
npm run build
```
