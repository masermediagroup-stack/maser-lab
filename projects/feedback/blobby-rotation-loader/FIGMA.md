# Figma / visual reference — Blobby Rotation Loader

No Figma file. Reference is a mobile playground screenshot set.

## Layout

- Full black background
- Centered loader (~40–50% viewport)
- Title "Blobby Rotation" centered top
- Circular back button top-left
- Four sliders fixed bottom with safe-area padding

## Slider styling

- Label + value on left: `Blur : 7`
- Yellow active track, dark grey inactive track
- Large white rounded-rectangle thumb

## Color defaults

- Core: white `#ffffff`
- Warm fringe: orange `#ff8c42`
- Cool fringe: blue `#4da6ff`
