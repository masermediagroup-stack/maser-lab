# Maser-Lab Three.js Command

Start or continue Three.js, shader, 3D, or advanced interaction work in Maser-Lab.

## Usage

`/maser-threejs [project-slug or brief]`

## Steps

1. Load [`.agents/skills/maser-lab-threejs/SKILL.md`](.agents/skills/maser-lab-threejs/SKILL.md)
2. Load [`.agents/skills/maser-lab-threejs/AGENTS.md`](.agents/skills/maser-lab-threejs/AGENTS.md)
3. Run **Skill Discovery** → `/find-skills` if skills gap
4. Run **Research agent** — [`.agents/skills/maser-lab-threejs/references/agents/research.md`](.agents/skills/maser-lab-threejs/references/agents/research.md)
5. Follow [workflow](.agents/skills/maser-lab-threejs/references/workflow.md)

## Authority

- [Official Three.js docs](https://threejs.org/docs/) — API truth
- [CloudAI-X threejs-skills](https://github.com/cloudai-x/threejs-skills) — patterns (verify against docs)

## Deep reasoning

`/poteto-mode` — **needs confirmation**. Until confirmed, use **Shape + Research agent** for multi-unknown tasks.
