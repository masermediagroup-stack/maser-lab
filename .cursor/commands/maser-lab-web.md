# Maser-Lab Web Command

Start or continue web UI work in Maser-Lab — page sections, components, forms, navigation, scroll reveals, and micro-interactions.

## Usage

`/maser-lab-web [project-slug or brief]`

## Steps

1. Load [`.agents/skills/maser-lab-web/SKILL.md`](.agents/skills/maser-lab-web/SKILL.md)
2. Load [`.agents/skills/maser-lab-web/AGENTS.md`](.agents/skills/maser-lab-web/AGENTS.md)
3. Read [`AGENTS.md`](AGENTS.md) for repo scope and layout
4. If a slug is in scope, read `projects/{category}/{slug}/PROJECT.md`
5. Declare request mode: Shape, Implement, Review, Motion-review, Harden, or Transfer

## Related commands

- `/maser-threejs` — Three.js, shaders, WebGL/WebGPU
- `/loop` — iterative refinement
- `/loop-status` — progress check
- `/find-skills` — discover capabilities

With Figma references, also load `figma-design-workflow`.
